# McGill Frosh2022 Website

### Install Docker
https://docs.docker.com/desktop/mac/install/

### Install Node
https://nodejs.org/en/download/

or through homebrew:
brew install node

#### Update npm
sudo npm install npm --global

### Install Nest js
npm i -g @nestjs/cli

## Run Backend
Make sure Docker Desktop is open
cd backend
npm i
npm run start:dev

## Run Frontend
cd frontend
npm i
npm run start
